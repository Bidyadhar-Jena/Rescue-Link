# RescueLink AI 🚨

RescueLink AI is a smart emergency response prototype developed by Team Bug Slayers for InnovaHack Chapter 1. The platform enables users to instantly trigger SOS alerts with live location tracking during emergencies, helping reduce response time when every second matters.

## 🔥 Features

- 🚨 One Tap SOS Alert
- 📍 Live Location Tracking
- 🔐 User Authentication
- 🗄️ MongoDB Database Integration
- ⚡ Real-Time Backend Communication
- 🎨 Modern Responsive UI
- 🤖 Scalable Architecture for Future AI Integration

## 🧠 Problem It Solves

RescueLink AI solves the problem of delayed emergency response by enabling users to instantly send SOS alerts with live location tracking during accidents, medical emergencies, or critical situations.
